# Personal website.

This repository contains code to generate my personal website. Pages can be
written in markdown, and is automatically converted to static html.

>[!NOTE]
> Bespoke python library that is a dumber/simplified version of something like Jekyll.

## Usage:

  - ```python main.py``` to create static site.
  - Serve the contents of this folder.
  - ```assets/content/*.md``` update, run script ```python main.py```, and push the repo.
